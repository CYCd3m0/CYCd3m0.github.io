Chinese Remainder Theorem

$$ \begin{cases} x \equiv a_{1} \ ( \bmod \ m_{1} ) \\ x \equiv a_{2} \ ( \bmod \ m_{2} ) \\ \hspace{3.5em} \vdots \\ x \equiv a_{n} \ ( \bmod \ m_{n} ) \end{cases} $$

已知合數 $M = m_{1} \times m_{2} \times \cdots \times m_{n}$

$m_{1},m_{2}, \cdots ,m_{n}$ 為彼此互質的整數 ( $m_{i}$ 可為質數的 $k$ 次方，如 $m_{1} = 5^{3}$ )


我們可以透過 $a_{1},a_{2}, \cdots ,a_{n}$ 以及 $m_{1},m_{2}, \cdots ,m_{n}$ 計算出 $A$

符合 $x \equiv A \ ( \bmod \ M )$


Solution

$$ A = \sum_{i=1}^{n} a_{i} \times t_{i} \times M_{i} \ ( \bmod \ M) $$

$M_{i} = M / m_{i}$

$t_{i} M_{i} \equiv 1 \ ( \bmod \ m_{i} )$

Note

$t_{i}$ 是 $M_{i}$ 對同餘 $m_{i}$ 的模反元素

$t_{i} = \text{inverse(} M_{i} , m_{i} \text{)}$


Example

$$ \begin{cases} x \equiv 2 \ ( \bmod \ 11 ) \\ x \equiv 3 \ ( \bmod \ 17 ) \\ x \equiv 3 \ ( \bmod \ 19 ) \end{cases} $$

$m_{i} = [11, 17, 19]$

$M_{i} = M/m_{i} = [323, 209, 187]$

$t_{i} = \text{inverse(} M_{i} , m_{i} \text{)} = [3, 7, 6]$

$ A = [\ (2 \times 3 \times 323) + (3 \times 7 \times 209) + (3 \times 6 \times 187)\ ] \ \bmod 3553 \\ \hspace{1em} = 2587 $


$x \equiv 2587 \ ( \bmod \ 3553 )$


Example2

$$ \begin{cases} x \equiv 2 \ ( \bmod \ p ) \\ x \equiv 2 \ ( \bmod \ q ) \end{cases} \hspace{1.5em} p , q \hspace{0.5em} \text{互質} $$

$x = k_1 p + 2$

$x = k_2 q + 2$

$k_1 p + 2 = k_2 q + 2$

$k_1 p = k_2 q$


$\because p , q \text{互質}$

$\therefore k_1 = r_1 q$

$\hspace{1em} k_2 = r_2 p$

$\because k_1 p = k_2 q$

$\therefore r_1 = r_2$ ( call $r$ )


$x = rpq + 2$

$x \equiv 2 \ ( \bmod \ pq )$

from Crypto.Util.number import inverse
from functools import reduce

def solve_CRT(list_a, list_m):
    sum = 0
    M = reduce(lambda x, y: x * y, list_m)
    for ai, mi in zip(list_a, list_m):
        Mi = M // mi
        sum += ai * Mi * inverse(Mi, mi)
    return sum % M
