Elliptic Curve

$$ y^{2} = x^{3} + ax + b \\ \text{where} \hspace{1em} 4a^{3} + 27b^{2} \not = 0 $$

  • 單位元素 $0$ ，代表一個無窮遠的點

  • $P$ 的反元素 $-P$ ，代表 $P$ 對 $x$ 軸的鏡射

    • $P + (-P) = 0$

定義

$$ \lbrace (x,y) \isin \Bbb{R}^{2} \hspace{0.5em} | \hspace{0.5em} y^{2} = x^{3} + ax + b \hspace{0.5em} , \hspace{0.5em} 4a^{3} + 27b^{2} \not = 0 \rbrace \cup \lbrace 0 \rbrace $$

  • $(\Bbb{G},+)$ 是一個 Commutative Group

幾何加法

$$ P + Q = R \\ P , Q\ \text{連線，在橢圓曲線上的交點對} x \text{軸的鏡射即為}\ R $$

  • 如果 $Q = 0$

    • $P + 0 = P$

  • 如果 $Q = -P$

    • $P + (-P) = 0$

  • 如果 $Q = P$

    • $P + P = R$ ，橢圓曲線在 $P$ 點的切線與橢圓曲線的交點對 $x$ 軸的鏡射即為 $R$

  • 如果 $Q \not = P$ ，但 $P , Q$ 連線，在橢圓曲線上沒有交點

    • 如果切點為 $P$ ，則 $P + Q = -P$

    • 如果切點為 $Q$ ，則 $P + Q = -Q$


代數加法

$$ P = (x_{P},y_{P}) \hspace{2em} Q = (x_{Q},y_{Q}) \hspace{2em} R = (x_{R},y_{R}) $$

  • 如果 $P \not = Q$

    • 斜率 $m = \dfrac{y_{P}-y_{Q}}{x_{P}-x_{Q}}$

    • $x_{R} = m^{2} - x_{P} - x_{Q}$

    • $y_{R} = m(x_{P}-x_{R}) - y_{P}$

    • 即使 $P$ 或 $Q$ 其中一個是切點，也可以使用


  • 如果 $P = Q$

    • 斜率 $m = \dfrac{3x_{P}^{2}+a}{2y_{P}}$

    • $x_{R} = m^{2} - 2x_{P}$

    • $y_{R} = m(x_{P}-x_{R}) - y_{P}$


純量乘法

$$ nP = \underbrace{P + P + \cdots + P}_{n\ \text{times}} $$


Elliptic Curve Cryptography

  • The field of integers modulo $p$ 有以下表示方法

    • $\Bbb{Z}/p$

    • $GF(p)$

    • $\Bbb{F}_{p}$


  • $x/y\ $ in $\ \Bbb{F}_{p}$

    • $x/y = x \times y_{inv}$

    • $x$ 乘上 $y$ 的乘法反元素


定義 Elliptic curves in $\Bbb{F}_{p}$


$$ \lbrace (x,y) \isin (\Bbb{F}_{p})^{2} \hspace{0.5em} | \hspace{0.5em} y^{2} \equiv x^{3} + ax + b\ (\bmod\ p) \hspace{0.5em} , \hspace{0.5em} 4a^{3} + 27b^{2} \not \equiv 0\ (\bmod\ p) \rbrace \cup \lbrace 0 \rbrace $$


Elliptic curves in $\Bbb{F}_{p}$ 依然是 Commutative Group


代數加法

$$ P = (x_{P},y_{P}) \hspace{2em} Q = (x_{Q},y_{Q}) \hspace{2em} R = (x_{R},y_{R}) $$

  • 如果 $P \not = Q$

    • 斜率 $m = (y_{P}-y_{Q})(x_{P}-x_{Q})_{inv} \bmod p$

    • $x_{R} = (m^{2} - x_{P} - x_{Q}) \bmod p$

    • $y_{R} = \lbrack m(x_{P}-x_{R}) - y_{P} \rbrack \bmod p$


  • 如果 $P = Q$

    • 斜率 $m = (3x_{P}^{2}+a)(2y_{P})_{inv} \bmod p$

    • $x_{R} = (m^{2} - 2x_{P}) \bmod p$

    • $y_{R} = \lbrack m(x_{P}-x_{R}) - y_{P} \rbrack \bmod p$


其實跟 $\Bbb{R}^{2}$ 的算法完全一樣，只是多了 $(\bmod\ p)$


Order of an elliptic curve group

  • Elliptic curve group 中，點的數量就代表 group 的 order

Subgroups

  • Subgroup

    • subgroup 中，點的數量依然代表 subgroup 的 order

    • cyclic subgroup 中，$P$ 的 order ，是使 $nP = 0$ 的最小正整數 $n$

    • group 的 order 必為其 subgroup order 的整數倍

    • ECC 中，我們需要一個 order 很大的 subgroup

      • 並不是先選擇 base point (generator) 再計算 order

      • 先計算 group 的 order $(N)$ ，再選擇足夠大的 subgroup order $(n)$

        • $NP = 0$ ，因為 $N$ 是 $n$ 的整數倍
