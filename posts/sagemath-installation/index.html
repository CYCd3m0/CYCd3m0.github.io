SageMath 9.1 on OSX

SageMath 從 9.0 版，開始用 python3

重裝的同時，紀錄下過程


SageMath

  • 從官網下載 SageMath 9.1

    • 我是下載這裡的 sage-9.1-OSX_10.11.6-x86_64.app.dmg

  • 下載完就點開，把檔案丟進 /Applications

  • 最後加個捷徑 

    1

    sudo ln -s /Applications/SageMath-9.1.app/sage /usr/local/bin/sage


完成就能直接執行 sage 檔，或使用 sage shell

1

sage <SageFile>

SageMath with Python

  • 如果要在 sage 裡寫 python

    • 就在 python 的檔案加上 

      1

      from sage.all import *

    • 並且用 sage 的 python 去執行 

      1

      sage -python <PythonFile>


  • sage 的 python 是獨立的 (所以要重新 pip install 一堆 package)

    • 使用 sage 的 pip 安裝 

      1

      sage --pip install <package>

    • 以 request 為例 

      1

      sage --pip install request

    • 以 pwntools 為例 

      1

      sage --pip install git+https://github.com/Gallopsled/pwntools.git
